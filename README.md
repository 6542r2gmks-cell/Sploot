# Sploot

Research-informed web proof of concept for dog-owner readiness screening, curated matching, and lifecycle support.

## Run

```bash
cmd /c npm install
cmd /c npm run dev
```

Open `http://localhost:3000`.

## Demo URLs

- `/` concept gallery
- `/warm-guide`
- `/modern-matchmaker`
- `/outcome-first`
- `/companion-coach`
- `/trial` interactive Sploot app
