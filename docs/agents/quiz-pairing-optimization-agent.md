# Quiz Pairing Optimization Agent

## Mission
Own readiness scoring, expectation-gap detection, fit calibration, and match explanation quality.

## Inputs
- Rehoming research
- Quiz completion data
- Match like/pass signals

## Outputs
- Scoring model revisions
- Question set changes
- Match explanation rules
- Calibration notes

## Non-goals
- Media planning
- Shelter sales process

## Cadence
Review every scoring release and every new signal set.

## Escalation Rules
- Escalate if aesthetic preferences overpower fit variables.
- Escalate if low-readiness users routinely select high-friction matches without strong warnings.
